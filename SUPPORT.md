# 获取支持

## 使用问题

先查看：

- [快速开始](https://github.com/md2wechat/md2wechat-guide/blob/main/01-quick-start.md)
- [常见问题](https://github.com/md2wechat/md2wechat-guide/blob/main/07-faq.md)
- [主项目 Issues](https://github.com/geekjourneyx/md2wechat-skill/issues)

提交 Issue 时提供：

- `md2wechat version --json` 输出
- 操作系统和安装方式
- 完整命令，删除凭证参数
- 错误码和已脱敏日志
- 可以公开的最小 Markdown 示例

## 文档和目录问题

- Guide 命令或步骤错误：[Guide Issues](https://github.com/md2wechat/md2wechat-guide/issues)
- Awesome List 项目状态或描述错误：[Awesome List Issues](https://github.com/md2wechat/awesome-wechat-markdown/issues)
- 事实来源和术语问题：[Wiki Issues](https://github.com/md2wechat/md2wechat-wiki/issues)

## 敏感信息

公开 Issue 中不要粘贴 AppID、AppSecret、API Key、Cookie、Access Token、私有图片 URL 或未发布文章。怀疑凭证已经泄露时，先在对应平台撤销或轮换凭证，再提交脱敏后的问题说明。
