## md2wechat 是做什么的？

`md2wechat` 是给 Agent 和内容团队用的公众号发布引擎。

你给它一份草稿、一个选题，或者一段初步想法，它会产出一版可审、可改、可发的稿子。

它负责四件事：

- 帮你补齐文章结构
- 帮你对齐作者语气
- 帮你补上转化模块
- 帮你把草稿推进到发布前状态

## 你买到的是最后一公里交付能力

| 方式 | 能做到什么 | 做不到什么 |
|---|---|---|
| 自己手动写 | 最懂自己 | 慢，容易卡在结构和收尾 |
| 免费排版工具 | 让文章更整齐 | 解决不了语气、结构、转化 |
| **md2wechat** | **把草稿变成可发布稿** | **交付结构、语气和转化闭环** |

## 用户第一次会看到什么

- 开头、转场、证据、结尾、CTA 一次补齐
- 样式、表达、转化一起处理
- 输出直接进入审稿和发布流程
